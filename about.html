About This Budget Tracker

The Budget Tracker is a personal finance web application that helps users track their income and expenses in a simple and organized way.

It allows users to add transactions with descriptions, categories, and dates. The system automatically updates the total balance in real time.

All data is stored in the browser using local storage, meaning your information remains saved even after refreshing or closing the page.

💰 Key Features

  • Add income and expense transactions
  • Track transactions by date
  • Categorize expenses (Food, Transport, Rent, etc.)
  • View real-time balance updates
  • Visualize data using charts (pie & bar graphs)
  • Delete incorrect transactions

📊 What Makes This Project Special

This project demonstrates practical JavaScript skills including DOM manipulation, event handling, data persistence, and data visualization using charts.

It is designed as a real-world style application, similar to simple financial tracking tools used for budgeting and expense management.

⚙️ Technologies Used

  • HTML5 for structure
  • CSS3 for styling and layout
  • JavaScript for logic and interactivity
  • Chart.js for data visualization
